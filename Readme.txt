Pour éviter l'actualisation du site quand les projets sont ajoutés ou supprimés vous devez au préalable dissocier le dossier FrontEnd du dossier Backend qui se situe dans : Portfolio-architecte-sophie-bluel-Hautefaye.

Vous pourrez par la suite lancer le WindowsLiveServer depuis  index.html 

Si vous ne dissociez pas les deux dossiers, vous devez modifier la ligne 52 du fichier login.html en indiquant le chemin suivant : "../FrontEnd/login.html" toute fois l'ajout et la suppression des projets entrainera une actualisation de la page.
